# Precover
[![Maven Central](https://img.shields.io/maven-central/v/io.github.donald-okara.precover/core)](https://central.sonatype.com/artifact/io.github.donald-okara.precover/core)

Precover is a static analysis and reporting tool for Jetpack Compose that evaluates preview coverage quality across themes, font scales, and screen sizes.

## 📖 Documentation
Detailed documentation can be found in the [docs/](docs/README.md) directory:
- [Getting Started](docs/users/GETTING_STARTED.md)
- [Built-in Rules](docs/users/RULES.md)
- [@PrecoverLink Guide](docs/users/PRECOVER_LINK.md)

## 🤖 AI Integration
Precover is designed to work seamlessly with AI coding assistants. We provide a set of "skills" (prompts and context) to help AI agents assist you in improving your UI coverage.

Check out the [AI Skills](skills/README.md) directory for more information.

## 🚀 Key Features
- **Automated Coverage Analysis**: Detects missing previews for different themes, font scales, and devices.
- **Scenario Tracking**: Ensure your components are tested across all UI states (Loading, Error, Success).
- **Baseline Workflow**: Maintain and improve coverage over time with baseline checks.
- **Aggregate Reporting**: Single dashboard for multi-module projects.
