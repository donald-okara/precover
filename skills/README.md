# Precover AI Skills

This directory contains resources to help AI coding assistants understand and interact with the Precover ecosystem. These "skills" provide the necessary context and guidelines for an AI to help you improve your Jetpack Compose preview coverage.

## Available Skills

- [**Compose Previews & Precover**](compose-previews.md): Guidelines for writing previews that satisfy Precover rules (Themes, Font Scales, Scenarios, etc.).

## How to use

You can provide these documents as context to your AI assistant (like GitHub Copilot, ChatGPT, or Claude) to enable them to give better suggestions for your UI testing.

For example, you can tell your AI:
> "Read `skills/compose-previews.md` to understand how to improve my Compose Preview coverage. If you need more technical details, refer to the documentation in the `docs/` folder."
